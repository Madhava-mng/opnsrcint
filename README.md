# Opnsrcint

OSINT tool for searching username
written in ruby

## Installation

```ruby
gem 'opnsrcint'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install opnsrcint

## Usage

		$ opnsrcint [cmd] [argv]
